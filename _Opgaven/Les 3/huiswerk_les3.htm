Aptana Editor

Benefits
  • Same keyboard commands as Eclipse
  • You can run individual projects
  • You're not really limited by language (I do everything from basic HTML/CSS and JavaScript to my .MD files for GitHub to Ruby on Rails)
  • It has a nice file system setup so you can easily find what you're doing.

To do web development in Eclipse, you should first go to the marketplace and get WTP, aka the Web Tool Platform. This plug-in offers a lot of very useful functionality. But it's not perfect. The WTP is missing a few things that you'll quickly notice as you do your work.
